# Todo React App with Redux and TypeScript
Todo React App with Redux, TypeScript, webpack 

## Description
Minimum Todo React App to create tasks and check them as Google Keep.


[Live demo](http://useful-todo.surge.sh/)

## Setup & Run

```
$ npm install
```
```
$ npm start
```
